# Taches
A simple to-do list app made using React Native. Visit [here](https://expo.dev/@tyagidemon/taches) to check this out.

## Getting Started
- Clone the repository on your computer.
- Make sure you have expo cli installed globally on your computer. If not, you can refer [here](https://expo.dev).
- Run `npm install` or `expo install` to install all the dependencies.
- Run `npm start` to start the project. Keep an eye on terminal for further instructions to run on Android/IOS device/emulator or web.
